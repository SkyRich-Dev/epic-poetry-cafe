4.6 Fixed price + discount handling
Need to support:
fixed selling price
item-wise discount
invoice-level discount
discounted unit value
final value per item
Logic
Default fixed price comes from menu item price.
If invoice-level total discount is provided but line discount is not:
allocate discount proportionally based on line gross value.
Formula
line gross = qty × fixed price
allocated line discount =
(line gross / invoice gross) × invoice total discount
final line amount = line gross - allocated line discount
Store this at line level.
